COMO ATIVAR A API-FOOTBALL GRATUITA

1. Cria a tua conta/key gratuita na API-Football / API-Sports.
2. Abre o ficheiro script.js.
3. Procura esta linha:

const API_FOOTBALL_KEY = 'COLOCA_AQUI_A_TUA_KEY';

4. Substitui COLOCA_AQUI_A_TUA_KEY pela tua key gratuita.

Exemplo:
const API_FOOTBALL_KEY = 'abcd1234...';

A app fica assim:
1) API-Football = prioridade para estado real do jogo, minuto, resultado, fim do jogo e eventos/golos.
2) worldcup26.ir = backup.
3) TheSportsDB = backup e lineups quando existir idEvent.
4) matches.json/Firebase = fallback local.

Nota: o plano gratuito da API-Football costuma ter limite diário. Para poupar pedidos, a app não chama esta API a cada 30 segundos; ela tem um intervalo mínimo próprio de 90 segundos.

CORREÇÃO — Zafronix WC API na Tabela dos Grupos

O que foi adicionado:
- Zafronix WC API configurada com a key fornecida.
- Endpoint usado: /standings?year=2026.
- A secção JOGOS > Tabela dos Grupos passa a usar a classificação oficial da Zafronix quando a API responder.
- A tabela agora mostra: posição, equipa, J, V, E, D, GM, GS, DG e pontos.
- Se a Zafronix falhar por quota/CORS/rede, a app continua a usar a tabela calculada por resultados/Firebase como fallback.

Nota:
Como o projeto é frontend estático, a key fica visível no browser. Para produção, o ideal é usar um proxy/backend teu para esconder a key.

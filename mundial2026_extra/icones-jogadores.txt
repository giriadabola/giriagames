Palavras disponíveis para o campo "icon" no Firebase:

lion  = Leão
wave  = Água
buffalo = Búfalo
elephant = Elefante
tiger = Tigre
hyena = Hiena
boa = Jiboia
gator = Jacaré
hippo = Hipopótamo
eagle = Águia
wolf = Lobo
rhino = Rinoceronte
cobra = Cobra
shark = Tubarão
falcon = Falcão
panther = Pantera
bear = Urso
fox = Raposa
owl = Coruja
bull = Touro

Exemplo de uso no documento do jogador:
icon: "lion"

Podes usar também as palavras em português sem acentos, por exemplo:
leao, agua, bufalo, jacare, hipopotamo, aguia, tubarao, falcao.

Nota: agora os ícones só aparecem se o documento do jogador/prognóstico tiver o campo icon no Firebase. O site já não escolhe animais aleatórios.

ALTERAÇÃO FEITA

Adicionei uma correção no script.js para tentar buscar o minuto live pela ESPN hidden API:

https://site.api.espn.com/apis/site/v2/sports/soccer/fifa.world/scoreboard?dates=YYYYMMDD

O código agora:
- procura as datas relevantes dos jogos;
- consulta a ESPN por data;
- lê o relógio em competition.status.displayClock;
- transforma valores como 23', 45'+2' ou 23:12 em minuto simples;
- dá mais prioridade a uma fonte que tenha minuto real em vez de apenas "live";
- mantém as outras APIs como fallback.

FICHEIRO PRINCIPAL ALTERADO

script.js

FICHEIRO EXTRA INCLUÍDO

espn-proxy-worker.js

Só precisas do espn-proxy-worker.js se o browser bloquear a ESPN por CORS.

COMO TESTAR

1. Faz upload destes ficheiros para o teu projeto.
2. Abre a app.
3. Quando houver jogo live, vê se o card mostra minuto real, por exemplo 23 ou 45+2.

SE A ESPN FALHAR POR CORS

1. Cria um Cloudflare Worker.
2. Cola o conteúdo de espn-proxy-worker.js.
3. Publica.
4. No script.js muda:

const ESPN_PROXY_URL = '';

para:

const ESPN_PROXY_URL = 'https://o-teu-worker.workers.dev';

IMPORTANTE

A ESPN é uma API não oficial/hidden. Pode mudar ou falhar. Mas não usa a quota da API-Football.

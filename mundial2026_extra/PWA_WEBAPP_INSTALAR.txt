PROTOCOLO WEB APP / PWA

Link público configurado:
https://giriagames.com/mundial2026_extra/index.html

Ficheiros adicionados/alterados:
- index.html
  Inclui o banner de instalação e o script pwa-install.js.

- site.webmanifest
  Define nome, ícones, start_url e scope da web app.

- sw.js
  Service Worker necessário para o browser reconhecer a app como instalável.

- pwa-install.js
  Controla o banner, o botão Descarregar, o fecho do banner e instruções para iPhone/iPad.

- style.css
  Estilos do banner.

Notas:
- Em Chrome/Edge/Android, o botão “Descarregar” usa o evento nativo de instalação quando disponível.
- Em iPhone/iPad, a Apple não permite instalação automática; o banner explica para usar Partilhar > Adicionar ao Ecrã principal.
- O site tem de estar em HTTPS, como giriagames.com, para o PWA funcionar.

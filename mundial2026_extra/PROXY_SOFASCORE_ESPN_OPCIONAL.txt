SOFASCORE E ESPN

Foram adicionados como fallback extra.

IMPORTANTE:
- São fontes não oficiais/hidden/scraping.
- Em alguns browsers podem falhar por CORS.
- Se isso acontecer, a app passa automaticamente para a próxima fonte.
- Para ficar mais fiável, podes usar um proxy teu.

No script.js existem estes campos:

const SOFASCORE_PROXY_URL = '';
const ESPN_PROXY_URL = '';

Exemplo:
const SOFASCORE_PROXY_URL = 'https://teusite.com/api/proxy';
const ESPN_PROXY_URL = 'https://teusite.com/api/proxy';

O proxy deve receber ?url=URL_DA_FONTE e devolver o JSON.
